Meme What You Say

Meme What you say is a real time multiplayer iOS app. The game is inspired by the popular games What Do You Meme and Psych!

The backend of the app was made using firebase which provided the Cloud Firestore database, and Cloud Storage bucket for photos. The frontend was built using React Native and Expo. While Expo allows deployment to both android and ios, the app was styled with iOS in mind however deployment to android is part of the apps future plans.

The Game:

  • One person hosts a game, generating a unique room code that they can share with their friends.
  • The host's friends then join the room and the host can start the game when everyone is ready
  • Once the game has begun, you will see a picture, the objective is to come up with the funniest caption.
  • After Each round, every player votes for their favourite (excluding their own)
  • after a number of rounds (that the host has chosen) the game is over and the player with the most votes is declared the winner!
